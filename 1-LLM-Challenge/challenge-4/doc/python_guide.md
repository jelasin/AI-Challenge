# Python编程指南

## 基础语法

Python 是一种高级、解释型语言，以简洁著称。

### 变量和数据类型

- int / float / str / bool
- list / dict / set / tuple

### 控制结构

```python
if cond:
    ...
elif other:
    ...
else:
    ...
```

## 面向对象编程

```python
class MyClass:
    def __init__(self, value):
        self.value = value
    def double(self):
        return self.value * 2
```
